#  Natural Language to Robot Motion

A Python-based 3D robotic arm simulation controlled by plain English commands.

Type: *"pick up the red cube and place it left of the blue one"*  
Watch: A 6-DOF robotic arm think, plan, and execute the motion in real-time.

## Tech Stack
- **PyBullet** — 3D physics simulation
- **Groq LLaMA 3** — Natural language AI brain
- **Inverse Kinematics** — Smooth, realistic arm movement
- **tkinter** — Control panel GUI
